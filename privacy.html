filecp Privacy Policy

Last updated: April 2026

Overview

filecp is a file sharing application that prioritizes your privacy. This policy explains what data we handle and how we protect it.

What We Collect

filecp does not collect, store, or transmit any personal information. Specifically:

  • No user accounts or registration required
  • No analytics or usage tracking
  • No cookies (tracking or otherwise)
  • No advertising identifiers
  • No device fingerprinting
  • No telemetry data

File Data

Files you upload through filecp are:

  • Encrypted at rest on our server using Fernet symmetric encryption
  • Stored temporarily for the duration of your session (which you control, from 1 minute to 24 hours)
  • Permanently deleted when the session expires — no backups, no archives
  • Not inspected, scanned, or analyzed in any way beyond what is necessary for the transfer

Network Communication

The filecp desktop application communicates only with our backend server (flitto.onrender.com) to facilitate file transfers. No data is sent to any third party, advertising network, or analytics service.

Local Storage

When you download files through the desktop app, they are saved to your local Downloads folder. The application does not access, read, modify, or delete any other files on your device.

Microsoft Edge WebView2

The desktop application uses Microsoft Edge WebView2 to render its user interface. WebView2 is a system component maintained by Microsoft and may have its own data handling practices as described in the Microsoft Privacy Statement.

Children's Privacy

filecp does not knowingly collect any information from children under the age of 13. The application is rated for ages 3+ as it contains no objectionable content.

Data Security

We take reasonable measures to protect your data during transit and at rest:

  • All file transfers use HTTPS encryption in transit
  • Files are encrypted at rest using Fernet (AES-128-CBC + HMAC-SHA256)
  • Encryption keys are generated per server instance and never stored persistently
  • Session data is stored in memory only and cleared on server restart

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date. We encourage you to review this policy periodically.

Contact

If you have questions or concerns about this privacy policy, please open an issue on our GitHub repository.
