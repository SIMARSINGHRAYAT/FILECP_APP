# FileCP Installer Return Codes Documentation

This page outlines the possible return codes from the FileCP installer and their meanings. If the installation encounters any issues, these codes will help you understand the outcome.

### Return Code List

- **0** – Installation successful: The installation completed successfully with no errors.
- **1602** – Installation cancelled by user: The user aborted the installation before it was completed.
- **1638** – Application already exists: The same version of FileCP is already installed on the device.
- **1618** – Installation already in progress: Another installation is currently running. Complete or cancel it before proceeding.
- **112** – Disk space is full: The installation stopped due to insufficient disk space on the device.
- **3010** – Reboot required: The installation finished, but a restart is required to complete the process.
- **12029** – Network failure: The installation failed due to a network issue (e.g., connection lost).
- **1625** – Package rejected during installation: A security policy on the device blocked the installation.

For additional installer troubleshooting or updates, check this repository or contact support.
