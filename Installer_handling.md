Title: Installer Return Codes for FileCP

Introduction:
This document details the return codes generated by the FileCP installer. These codes indicate various installation outcomes or errors. Testers and users can refer to this guide to understand the meaning of each code.

Return Code List:

0 – Installation successful: The application installed correctly without errors.
1602 – Installation cancelled by user: The user chose to cancel the installation before it completed.
1638 – Application already exists: The installer detected that the same version of the application is already installed.
1618 – Installation already in progress: Another installation is ongoing. The user must finish or cancel that process first.
112 – Disk space is full: The installer could not proceed due to insufficient disk space.
3010 – Reboot required: The installation succeeded, but a restart is needed to complete the process.
12029 – Network failure: The installation failed due to a network-related error.
1625 – Package rejected during installation: A security policy prevented the installation.

If other specific return codes are introduced, this page will be updated accordingly.
